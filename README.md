# Landing Just Solutions × GAMO — versión un solo archivo

Esta versión está preparada para evitar errores de rutas con CSS, JavaScript o imágenes.

## Cómo subir a GitHub Pages

1. Entra en el repositorio `justsolutions-seo-geo`.
2. Borra los archivos actuales o crea un repositorio nuevo.
3. Sube únicamente `index.html` a la raíz del repositorio.
4. En Settings > Pages deja `main` y `/root`.
5. Espera 1-3 minutos y abre la URL pública.

No necesita carpeta assets, styles.css ni script.js porque todo va integrado dentro del HTML.
